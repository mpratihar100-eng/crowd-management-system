# Crowd Management System

Production-ready C-based computer vision system for real-time crowd monitoring.

## Features

- Real-time person detection and tracking
- Spatial heatmap generation
- Automated staff recommendations
- Kubernetes-ready with autoscaling
- Full observability stack
